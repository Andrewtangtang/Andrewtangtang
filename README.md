## About Me

- Senior undergraduate Computer Science student at National Cheng Kung University (NCKU), Taiwan.
- Research Assistant at [Secure Systems Lab](https://liswei.sh/sslab/), National Taiwan University, under Prof. Shih-Wei Li.
- Research focus: para-virtualization (virtio) and system performance optimization for Confidential Computing (CVMs), with ongoing work analyzing and improving the virtio-vsock communication interface.
- Interested in database storage systems, currently building a series of DuckDB extensions to explore storage internals and observability.
- Strong interest in virtualization, Linux kernel, and system-level performance engineering.

## Open Source Contributions

- [Moonlink](https://github.com/Mooncake-Labs/moonlink)  
  Iceberg-native ingestion engine enabling streaming inserts and upserts for lakehouse architectures.
  - Supports ingestion from Postgres CDC, Kafka event streams, and OTEL telemetry data.
  - Uses buffering, caching, and indexing to keep Iceberg tables read-optimized without heavy maintenance or compaction.

- [DuckDB Extensions](https://github.com/duckdb/community-extensions)
  - [duckdb-table-inspector](https://github.com/dentiny/duckdb-table-inspector)  
    DuckDB extension providing observability into DuckDB storage internals.
    - Inspect storage usage at database, table, and column levels.
    - Diagnose issues such as unexpected database file growth and inefficient compression.
  - Coming soon...
